A custom Gutenberg block to display posts with advanced filtering options.

== Description ==

Glimpse Post Block is a powerful Gutenberg block that allows you to display posts with advanced filtering and customization options.

= Features =

* Filter posts by category
* Filter posts by tags
* Select specific posts to display
* Control number of posts to show
* Show/hide post title
* Show/hide featured image
* Show/hide excerpt
* Choose link placement (title or button)
* Fully responsive design
* Translation ready

= Security =

This plugin follows WordPress coding standards and security best practices.

== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/glimpse-post-block` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Use the 'Glimpse Posts' block in the Gutenberg editor.

== Frequently Asked Questions ==

= How do I use this block? =

Simply add the "Glimpse Posts" block to your post or page and configure the settings in the block sidebar.

= Can I use this with custom post types? =

Currently, the block only works with standard posts. Custom post type support may be added in future versions.
